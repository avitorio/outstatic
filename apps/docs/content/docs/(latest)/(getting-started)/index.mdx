## Quick start

The fastest way to see how Outstatic works is to use the hosted dashboard.

[Create an account](https://outstatic.com/auth/sign-up?provider=github&utm_source=github&utm_medium=readme&utm_campaign=quick_start) to start managing Markdown/MDX in your GitHub repositories without installing anything.
