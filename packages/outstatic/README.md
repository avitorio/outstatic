# Outstatic

Static CMS for Markdown and MDX, with content stored in GitHub.

[Website](https://outstatic.com/?utm_source=github&utm_medium=readme&utm_campaign=repo_nav) · [Docs](https://outstatic.com/docs/introduction?utm_source=github&utm_medium=readme&utm_campaign=repo_nav) · [Try it](https://outstatic.com/auth/sign-up?provider=github&utm_source=github&utm_medium=readme&utm_campaign=repo_nav)

Outstatic gives you a visual dashboard for creating, editing, and publishing Markdown and MDX content. Your content is automatically committed to your GitHub repository and deployed with your website.

No database. No hosted CMS backend. No complicated setup.

👉 [Try Outstatic in your browser](https://outstatic.com/auth/sign-up?provider=github&utm_source=github&utm_medium=readme&utm_campaign=hero_cta)

## Quick start

The fastest way to see how Outstatic works is to use the hosted dashboard.

[Create a free account](https://outstatic.com/auth/sign-up?provider=github&utm_source=github&utm_medium=readme&utm_campaign=quick_start) and start managing Markdown in your GitHub repositories without installing anything.

Prefer to self-host?

[Follow the getting started guide](https://outstatic.com/docs/?utm_source=github&utm_medium=readme&utm_campaign=self_hosting) to add Outstatic to your own Next.js project.

## Key features

- 📝 Visual dashboard for editing Markdown and MDX
- 💾 Commits content directly to your GitHub repository
- 🚀 Works without a database or external CMS backend
- 🏠 Self-host with your existing Next.js app
- 🧩 Add custom fields for structured content
- 🤖 Use AI completions for faster writing

## Who is Outstatic for?

Outstatic is useful if you want:

- A CMS for a Markdown or MDX website
- Content stored in GitHub instead of a database
- A dashboard your team can use without editing files manually
- A simple way to manage blog posts, docs, changelogs, landing pages, or resource libraries
- A CMS that works well with static sites and modern Next.js projects

[See how Outstatic works](https://outstatic.com/?utm_source=github&utm_medium=readme&utm_campaign=who_is_it_for)

## Self-hosting

Outstatic can be installed in your own Next.js app, so your content workflow stays close to your codebase.

Use the self-hosted setup if you want to:

- Keep full control of your app and deployment
- Store content as files in your repository
- Avoid managing a separate CMS database
- Let editors publish content without manually editing Markdown files

[Read the getting started guide](https://outstatic.com/docs/?utm_source=github&utm_medium=readme&utm_campaign=self_hosting_section)

## Tech stack

Outstatic is built with:

- [Next.js](https://nextjs.org/) - framework
- [TanStack Query](https://tanstack.com/query/latest) - state management and data fetching
- [GitHub](https://github.com/) - content storage and auth
- [Tiptap](https://tiptap.dev/) - text editor
- [OpenAI](https://openai.com/) and [Vercel AI Gateway](https://vercel.com/ai-gateway) - AI completions
- [Vercel AI SDK](https://sdk.vercel.ai/docs) - AI library
- [Tailwind CSS](https://tailwindcss.com/) - styles
- [Novel](https://novel.sh/) - AI completions and slash command
